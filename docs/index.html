Odyssey Project

Contexto

Nosso projeto está associado ao Projeto Odyssey - Platform for Automated Sensing in Archaeology que tem como objetivo desenvolver técnicas de tratamento de imagem e de inteligência artificial para detetar sítios arqueológicos a partir de dados de diferentes fontes, nomeadamente de imagens aéreas captadas utilizando drones, visando automatizar e complementar o trabalho humano.

Objetivo

O projeto teve como objetivo principal o desenvolvimento de uma aplicação móvel com a capacidade de anotar e validar, no terreno, os resultados dos algoritmos de aprendizagem usados para detetar ocorrências de interesse arqueológico.
